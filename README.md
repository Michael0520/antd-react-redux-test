## Clone

```shell
git clone git@github.com:Michael0520/QRCode-class.git
cd QRCode-class
pnpm i # If not install pnpm, please run: npm install -g pnpm
```

## Usage

### Development

click it http://localhost:3000

```shell
pnpm dev
```

### Build

```shell
pnpm build
```

### Check

```shell
pnpm lint
pnpm lint:stylelint
```
